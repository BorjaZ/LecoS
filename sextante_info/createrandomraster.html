Create random Landscape (Distribution)

Brief Description

Allows to create a random new raster layer based on a choosen statistical distribution. The can specifiy the extent of the resulting raster and the input parameters for the distribution.

Parameters

  • [param] Input distribution: User can choose between a Constant value, Random integer from a range of values, a Uniform, gaussian normal, exponential, Poisson, Gamma, Binomial, Geometric, Negative binomial, lognormal or Weibull distribution.

  • [param] Input extent: The extent of which the raster should be created

  • [param] Minimum / Alpha value: A choosen minimum, respectively alpha value.

  • [param] Maximum / Beta: A choosen maximum, respectively beta value.

  • [param] Mean / Number: A choosen mean value or the number or single number

  • [param] Standard Deviation / Probability: A choosen standard deviation or probability value (between 0 and 1)

  • [param] Cellsize: The cellsize of the resulting raster (default: 25)

  • [param] Output Raster: New random raster

Limitations

  • None known yet
