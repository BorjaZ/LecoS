Label Landscape patches

Brief Description

Conducts a connected component labeling (chessboard structure) of all raster cells with a given value.
The output is a rasterlayer containing all patches with an incremental number (zone).

Parameters

  • [param] Input raster: The raster layer to work on

  • [param] A landscape class: Which values should be used for labeling the patches

  • [param] Output Raster: Output result

Limitations

  • Very large raster layers might prevent this tool from running due to memory issues.
